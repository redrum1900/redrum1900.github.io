比较常用的git命令，“真“常用，基本满足工作学习使用，那么开整

![zhuangbi](http://img2.imgtn.bdimg.com/it/u=2996204179,2628677315&fm=21&gp=0.jpg)

新的文件夹需要初始化

```
    git init

```

查看文件变更

```

    git status

```

添加文件

```
    git add [filename]      //添加单个文件

    git add -a              //添加所有文件

```

提交变化

```
    git commit -m"注释"

    git commit --amend      //可以修改最后一次提交的信息.但是如果你已经push过了，那么其历史最后一次，永远也不能修改了。
```
